# An Australian EPG for your IPTV Player

With the help of ChatGPT, this project fills in the gaps by providing an EPG for FAST channels from 7plus, 9Now, and 10 Play that are missing or no longer maintained in Matt Huisman’s EPG, https://raw.githubusercontent.com/matthuisman/i.mjh.nz/refs/heads/master/all/epg.xml. 

#### EPG Link: https://raw.githubusercontent.com/Eurotrasz/australia-epg/refs/heads/main/au_epg.xml

In order to use it, you'd need to create a custom playlist and change the tvg-ids to match the tvg-ids in my EPG, and point your IPTV player to that playlist instead.

If you're looking for something that's ready to go and has all the Australian free-to-air TV channels, as well as their FAST channels, you can use my playlist instead.

This file will get updated every 4 hours or so as I run the scripts on my server, and upload the .xml file to Github after each run. Set your IPTV player to update every few hours or so to get the latest EPG data as some of the Aussie FAST channels don't give much future programming info, so you might get "No Information" in the future if you aren't as updated.

## IPTV Player

If you're after a decent IPTV player, I recommend using Sparkle TV. It's available for Android so any Android-based TV, Google streaming device, Android emulator (Bluestacks, MSI Player... which is still Bluestacks, etc) and Fire TV (via sideloading it). 

I've made the playlist a single file as the free version of Sparkle TV only lets you run one file, but I've also put all the channels into multiple categories such as Comedy, Drama, TV Series, Movies, etv so one channel may appear in multiple categories, but it makes it a much better experience when flicking through channels.

It also calls all the relevant EPGs, which I've spent and continue to spend countless hours making sure almost every channel has EPG data for what they're showing, and all you have to do is point your IPTV player to this playlist file.

Fire TV is a relatively cheap streaming device to get and Amazon always has specials, so that's good to use if you don't have an Android-based TV. 

### Sparkle TV App
https://play.google.com/store/apps/details?id=se.hedekonsult.sparkle&hl=en_AU&pli=1

It works amazingly well and I've spent countless hours trying to get TV working with xTeve/Threadfin, Dispatchrr, etc and had constant loading and buffering issues.

In Sparkle TV, it's simple to setup, easy to use, updates well and just runs endlessly on Android-based devices. Fire TV hasn't been as steady for me compared to proper installed instances, but I'm not using the newest Fire TV devices.

Again, a suggestion on how to run it in Windows is to use an Android emulator. MSI Player seems to work well for me. I've run Sparkle in MSI Player on an old HP All-in-One PC with a Gen 7 Intel i5, with 8GB RAM and SSD, and it just works solid streaming 9-10 hours at a time.

Another good IPTV player app is TiViMate. I haven't realy used it myself but it's worth checking before purchasing anything. 

### MSI Player (Android Emulator)
https://www.msi.com/Landing/appplayer

I wouldn't recommend signing into your Google account unless you use a throwaway gmail to sign in. Instead, just open Chrome up on it, get the Sparkle TV app from an APK store like https://apkpure.com/sparkle-tv-iptv-player/se.hedekonsult.sparkle/download. It makes you a shortcut to Sparkle TV directly on your Windows desktop which is great for media PCs or streaming devices along with an air mouse remote.
